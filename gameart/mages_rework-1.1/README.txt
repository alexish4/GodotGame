Name:
  Mages Rework
Version:
  1.1
Description:
  Reworked versions of Mage Sprites by Sollision, created as
  drop-in replacements for Stendhal.
Sprite details:
  - Dimensions: 24x32 & 48x64
  - Orientation: orthogonal (N/E/S/W)
  - Animation: 3 frames per direction.
  - PNG images use indexed color (smaller files).
  - GIMP sources (.xcf) use RGB color.
